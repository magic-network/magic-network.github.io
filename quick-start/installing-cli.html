    • Installing CLI

    The Magic CLI can be installed in 2 easy steps.

    Requirements

    • macOS 10.6 or higher
    • Python 2.7 or higher

    Installation

    1. Clone the repository from GitHub:
    git clone https://github.com/magic-network/magic-cli.git
    1. Install the CLI:
    pip install magic-cli
