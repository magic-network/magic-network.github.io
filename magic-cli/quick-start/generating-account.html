    • Generating Test Account

    When starting the Magic CLI you'll have the option to generate a new Ethereum account:

    Select the option to create a new test Ethereum account and the Magic CLI will generate a new wallet for you to use on the network. Every account has a private key and a public key and are indexed by an address. They will be displayed on the screen so you can save them in a safe place.
